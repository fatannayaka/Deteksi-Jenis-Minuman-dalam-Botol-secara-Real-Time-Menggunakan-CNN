## Cara menggunakan project: ##

1. Download ZIP dataset yang telah teranotasi.
2. Buka link training dataset dan buka file di googlecollab.
3. Install library yg dibutuhkan (tinggal run saja kolom kode program pertama).
4. Upload dataset yang teranotasi tadi di "select files" pada kolom kode program kedua.
5. Run kolom kode program training dengan 50 epochs, lalu tunggu hingga selesai dengan estimasi sekitar 3 s/d 4 jam.
6. Run kolom kode program untuk menampilkan grafik hasil training, jika training epoch sudah selesai.
7. Run kolom kode program terakhir untuk menyimpan seluruh hasil training dan grafiknya.
8. Buka aplikasi Vscode, lalu buat folder baru dan buat file program format .py.
9. Salin kode program "link webcam realtime".
10. Lalu buka hasil training tadi dan cari file "best.pt", dan copy file tersebut ke folder yang sama dengan file .py tadi pada vscode.
11. Run kode program di vscode.
12. Selesai
